# Friends Bill Splitter

Friends Bill Splitter: Dynamic friends management, multi-currency bills, real-time balances, charts, QR payments. Pure HTML/CSS/JS, fully responsive w/animations.

## Features
- Add and delete expenses
- Real-time balances for A and B + single net balance
- Smooth animations and professional UI
- Responsive design
- Data persists in browser (localStorage)
- No setup required - pure HTML/CSS/JS

## Quick Start
1. Open `index.html` in any modern web browser.
2. Add expenses using the form.
3. Balances update automatically.

## UI
- Solid color palette: Navy, grays, green/red accents.
- Animations: Fade-ins, hovers, smooth updates.
